# marco-rubio-bible-tweets
How many times does Senator Marco Rubio tweet bible verses? (Shockingly, up to 30% of his tweets are like this.)

Run `main.py` and then the `graph` notebook. 
## Dependencies
- pandas
- twint
- matplotlib
- Python STL (tested on 3.8)
