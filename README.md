# marco-rubio-bible-tweets
How many times does Senator Marco Rubio tweet verses from the Bible? I was able to find out with Twitter scraping.

Run `main.py` and then the `graph` notebook. 
## Dependencies
- pandas
- twint
- matplotlib
- Python STL (tested on 3.8)
